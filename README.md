
Burger is a app reasted with MySQL, Node, Express, Handlebars and a homemade ORM (yum!).
1. Create/Order your own burger
2. Press Devour button to eat it.
3. Press un-Devour button to un-eat it.
4. Press Delete buttin to delete the burger from the list.
